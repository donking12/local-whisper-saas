# 🎙️ تطبيق التفريغ الصوتي المحلي (On-Device Audio Transcriber)  ## 📌 عن المشروع هذا المشروع عبارة عن منتج برمجيات مصغر (Micro-SaaS) مصمم لتقديم خدمة تفريغ صوتي (Speech-to-Text) بدقة عالية مع **حماية مطلقة للخصوصية**.  تتم معالجة الملفات الصوتية بالكامل داخل متصفح المستخدم باستخدام كرت الشاشة (GPU) الخاص به، مما يعني عدم رفع أي بيانات حساسة إلى خوادم خارجية، وتقليل تكاليف التشغيل (Server Costs) إلى الصفر.  ## 🚀 الميزات التنافسية * **أمان وخصوصية 100%:** لا يتم رفع الملفات السمعية لأي سيرفر سحابي. * **العمل دون اتصال (Offline-First):** بعد تحميل الموقع والنموذج لأول مرة، يمكن استخدام الأداة بدون إنترنت. * **صفر تكاليف معالجة:** الاعتماد على قدرات جهاز العميل لمعالجة الذكاء الاصطناعي بدلاً من استئجار خوادم سحابية باهظة. * **معالجة محلية للفيديو:** استخراج الصوت من ملفات الفيديو الكبيرة محلياً قبل تفريغها.  ## 🛠️ التقنيات المستخدمة (Tech Stack) * **الواجهة الأمامية:** Next.js (App Router) + Tailwind CSS. * **محرك الذكاء الاصطناعي:** `Transformers.js` v4 (Hugging Face). * **المعالجة المسرعة:** واجهة `WebGPU` (باستخدام نموذج Whisper-tiny / base بدقة `fp16`). * **معالجة الوسائط:** `FFmpeg.wasm` لاستخراج الصوت وتحويله إلى 16kHz Mono. * **إدارة التراخيص (Backend):** PocketBase للتحقق من خطط الاشتراك (Freemium). * **بيئة الاستضافة المستهدفة:** Cloudflare Pages (كموقع Static).  ## ⚙️ آلية العمل (Architecture Flow) 1. **الرفع محلياً:** يقوم المستخدم بإسقاط ملف صوتي أو فيديو. 2. **التجهيز (Preprocessing):** يقوم المتصفح بفصل الصوت، وتحويله إلى تردد 16000 هرتز باستخدام `AudioContext`. 3
